# alx-backend-python

# Python Generators – Task 1: Getting started

This project sets up a MySQL database `ALX_prodev` and seeds it with data from `user_data.csv`.

### Functions
- `connect_db()` – connects to MySQL server
- `create_database(connection)` – creates database if not exists
- `connect_to_prodev()` – connects to `ALX_prodev`
- `create_table(connection)` – creates table `user_data`
- `insert_data(connection, data)` – inserts CSV data into table
